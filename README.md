# vite-tahoe-ui

Bringing Apple's Liquid Glass design language from macOS Tahoe to the web.

**[Live Demo](https://chicunic.github.io/vite-tahoe-ui/)**

![Preview](https://img.shields.io/badge/status-alpha-blue) ![React](https://img.shields.io/badge/React-19-61dafb) ![License](https://img.shields.io/badge/license-MIT-green)

## Design Philosophy

Tahoe UI recreates the stunning visual language Apple introduced in macOS Tahoe — Liquid Glass. Built around translucent materials, soft lighting, and refined motion, interface elements appear to float above content with a sense of depth and clarity.

## Features

- **Liquid Glass Effect** — Authentic frosted glass materials that respond dynamically to background content
- **Complete Component Library** — Windows, buttons, inputs, menus, dialogs, and more
- **Native macOS Experience** — Traffic light controls, sidebars, toolbars — everything feels familiar
- **Dark Mode** — Automatic adaptation to system theme preferences
- **Modern Stack** — React 19 + TypeScript + Vite
